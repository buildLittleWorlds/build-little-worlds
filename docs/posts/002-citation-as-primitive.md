# Citation as Primitive

This is a scaffold, not a finished citation note.

A citation is the smallest world this project can build. It points to a source, but it also creates a tiny set of obligations:

- Verify the wording.
- Locate the passage in the argument.
- Name why the passage matters.
- Decide what kind of later writing it could support.

For a Rieff project, a citation primitive might begin with a passage from one of his major works, then surround it with a short apparatus:

- **Source:** the exact work, edition, and page still to be verified.
- **Term:** the concept the passage helps define.
- **Question:** the interpretive problem opened by the passage.
- **Use:** the kind of paragraph, paper, or teaching note this citation might later support.

The important move is to resist treating citation as decoration. A citation primitive should become a workbench. It should let a reader see what has been checked, what is inferred, and what remains unsettled.

The first pass of this blog will use placeholders where exact bibliographic details still need to be supplied. That is intentional. The goal is to show the shape of the scholarly object before pretending that the object is complete.
